# research

Alle Rechercheergebnisse von nuresearch als Markdown-Dokumente
(`YYYY-MM-DD-<thema>.md`). Automatisch befüllt durch die Paperclip-Agenten
gemäß der Arbeitsanweisung in [`../AGENTS.md`](../AGENTS.md).

Der verbindliche Rechercherahmen & Dokumentstandard (Gliederung,
Quellenqualität, Zitationsformat, Fakt/Annahme-Trennung, Health-Claims-/HWG-
Checkliste), auf dem jedes Dossier aufbaut, steht in
[`../standards/methodik.md`](../standards/methodik.md).
